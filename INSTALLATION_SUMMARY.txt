## 📞 Support & Troubleshooting

### Common Issues

**Q: Occupancy showing incorrect percentage**
- A: Clear browser cache
- Check that Booking.booked is updated on approval
- Verify database has recent bookings

**Q: Priority bookings not appearing**
- A: Verify booking.isPriority flag is set
- Check admin is in /admin/priority-review page
- Verify user role is 'admin' or 'staff'

**Q: Notifications not showing**
- A: Check /api/notifications endpoint returns data
- Verify user has viewing permissions
- Check NotificationCenter component is in Navbar

**Q: Smart suggestions not appearing**
- A: Verify facilityServiceId and date params are correct
- Check /api/smart/suggestions responds with data
- Verify SmartSuggestionPanel is in BookingPage
